# OCPP Schemas

This directory contains JSON schemas for OCPP 1.6, OCPP 2.0.1, and OCPP 2.1
messages, used for strict payload validation in `ocppsim`.
The schemas are organized by OCPP version and message type, following the
structure of the OCPP specifications.
